# 结果段落模板

本模板用于把 `05_analysis_output/tables/` 中的统计表和 `05_analysis_output/notes/` 中的结果摘要，整理成可直接写入 `06_paper_draft/sections/` 的中文“结果”段落。

## 结果接口定位

- 表格文件：`05_analysis_output/tables/table_XX_<content>.md` 或 `.tex`
- 论文接口：`05_analysis_output/export_for_paper/paper_table_XX_<content>.tex`
- 结果摘要：`05_analysis_output/notes/note_XX_<content>.md`
- 复现日志：`05_analysis_output/logs/run_XX_<content>.md`

- 对应表格名称：
- 对应分析名称：
- 推荐文件基名：`table_XX_<content>`
- 回答的研究问题：
- 样本量：
- 因变量：
- 核心自变量：
- 模型列设置：

## 中文结果段落

表 X 报告了[分析主题]的主结果。列（1）展示了[基准设定]，列（2）在此基础上加入了[补充设定]，列（3）为[完整模型或替代模型]。结果显示，核心解释变量 `X` 的系数为[数值]，并在[显著性水平]上显著，说明在控制[控制变量或固定效应]后，`X` 与 `Y` 之间存在[正向/负向/不显著]关系。就经济意义或实际意义而言，[简述效应大小、方向和现实含义]。因此，该结果[支持/不支持/部分支持]研究预期。

- 与理论预期或既有研究是否一致：

## 扩展段落接口

如果该表不是主结果表，而是补充结果表，可改写为：

- 稳健性结果：强调估计方向、显著性和结论稳定性
- 异质性结果：强调不同组之间的差异
- 机制结果：强调中介、渠道或调节方向
- 替代设定结果：强调结论是否依赖特定变量定义或样本口径

## 自检清单

- 数字是否来自 `05_analysis_output/tables/` 或 `05_analysis_output/notes/`
- 表格编号、表名和段落引用是否一致
- 是否清楚区分“证据”“解释”“结论”
- 是否避免把整张表逐列机械复述
- 是否能直接移动到 `06_paper_draft/sections/04_results.tex` 或相应章节
